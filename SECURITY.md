# Security

Si encontrás una vulnerabilidad, por favor reportala de forma privada a los mantenedores.

- Email: diegoezequielliberman@gmail.com
- Respuesta esperada: 72 horas hábiles para confirmar recepción.

No publiques exploits públicamente hasta que exista un parche o acuerdo sobre coordinación de divulgación.

Recomendaciones para colaboradores:

- No subir credenciales ni archivos `.env` al repositorio.
- Usá vaults o variables de entorno en CI/CD.
- Escaneá el repo con `git-secrets` o `trufflehog` antes de publicar.
